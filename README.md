### Zuno/Streamly

## Project Planning

A full-stack music streaming application built with the MERN stack that allows users to browse, search, and play songs while creating and managing custom playlists. The app features user authentication, a responsive music player, and full CRUD operations for playlist management.

## MVP User Stories
Authentication & User Management:

- As a new user, I want to create an account with email and password
- As a user, I want to log in to my account securely
- As a user, I want to log out from any device
- As a user, I want to stay logged in between sessions

Music Browsing & Discovery:

- As a user, I want to browse available songs on the platform
- As a user, I want to search for songs by title or artist
- As a user, I want to view song details (title, artist, duration, album art)
- As a user, I want to see a list of recently played songs
- As a user, I want to view popular/top charts
Basic Music Player:

- As a user, I want to play and pause songs
- As a user, I want to see which song is currently playing
- As a user, I want to see progress through the current song
- As a user, I want to adjust volume levels
- As a user, I want to skip to the next song in a list

Playlist Management:

- As a user, I want to create new playlists
- As a user, I want to add songs to my playlists
- As a user, I want to remove songs from my playlists
- As a user, I want to view all my created playlists
- As a user, I want to delete playlists I no longer want

## Stretch Goal User Stories
Enhanced Music Experience:

- As a user, I want to shuffle play songs in a playlist
- As a user, I want to repeat a single song or entire playlist
- As a user, I want to seek to specific parts of a song
- As a user, I want to create and share song/playlist links


Library Management

- As a user, I want to "like" songs to add them to my favorites library

- As a user, I want to view my listening history
